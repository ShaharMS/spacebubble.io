Welcome!

The Haxe Learning Center is a collection of tutorials, guides and docs for Haxe. Here, you can learn many things about Haxe, follow guides about features that other haxelibs offer, learn how to fix errors, and more!
This place is supposed to have everything - from the very basic, to the more advanced concepts.
If you think something is missing, and you want to add a page, visit this website's github repository and contribute.

Things to get you started:

Haxe Features Learn some cool features and tricks that Haxe Offers.
Haxe is great; learn how to use it!
Fix Errors And Problems Errors can be reallllly annoying, especially when you can't find any documentation on the internet.
Im aiming to collect solutions to every single error one can encounter, so you won't waste your time looking for solutions.
Externs Learn how to use external code, that isn't necessarily written in Haxe - in your Haxe Program.
This includes the basics of writing externs for different platforms, and how to write Wrappers.
