Welcome!

The Haxe Learning Center is a collection of tutorials guides and docs for Haxe. Here, you can learn many things about Haxe, as well as understand some programming concepts used in other languages.
This place is supposed to have everything - from the very basic, to the more advanced concepts.
If you think something is missing, and you want to add a page, visit this website's github repository and contribute.

Things to get you started:

Programming Basics Learn the basics of programming in Haxe. This includes the basic types, variables, loops etc.
This is relevant for other languages as well.
Coding Patterns Further understand Haxe code patterns and their syntax: Abstracts, Interfaces, Lambda Functions...
