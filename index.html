Little

Little is a programming language I wrote, and it's goal is to teach children how to program, in an easy and accessible way. You can write it's source code in whatever language you'd like to write in - english, spanish, hebrew or arabic - and you'd still be able to run it :)

as of today (25th April, 2023), The language's development has finished, and it supports pretty much everything any programming language support, with some extra features on top :)
