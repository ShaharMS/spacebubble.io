Vision API

class Queue<T>

package vision.ds

Represents a queue, as a doubly linked list.

Constructor

new()

Creates a new Queue with 0 elements.

use enqueue()/dequeue() to add/remove elements.

Variables

first:QueueCell<T>

last:QueueCell<T>

@:value(0)read onlylength:Int = 0

Methods

dequeue():T

Gets the value of the first item in the queue, and removes it from the queue.

(last -> ... -> first)

enqueue(value:T):T

Adds a value to the end of the queue, and returns it.

Parameters:

value

The value to push to the end of the queue

has(value:T):Bool

Whether or not this Queue contains value.

This function iterates over the elements from the end of the queue to the start, and checks for equality using standard equality (==).

Parameters:

value

the value to check with

Returns:

Whether or not value is present in this Queue.

iterator():Iterator<T>

Returns an iterator over the elements of this Queue.

toString():String

Returns a String representation of this Queue, for example:

var queue = new Queue<Int>();
queue.enqueue(5);
queue.enqueue(7);
queue.enqueue(4);
trace(queue.toString());

Outputs:

[4 -> 7 -> 5]
