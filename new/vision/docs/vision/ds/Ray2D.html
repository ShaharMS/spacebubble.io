API Documentation

class Ray2D

package vision.ds

@:directlyUsed

Represents a 2-dimensional ray on the cartesian coordinate system

Static methods

staticinlinefrom2Points(point1:Point2D, point2:Point2D):Ray2D

Constructs a Ray2D from 2 Point2Ds

Parameters:

point1

First reference point, will be stored in the returned Ray2D's point field.

point2

Second reference point, used to calculate the slope of the ray.

Constructor

inlinenew(point:Point2D, ?m:Float, ?degrees:Float, ?radians:Float)

Parameters:

point

a point on the line

m

the slope of the line, optional - you can pass the degrees/radians instead, and set this to null

degrees

the degrees of the line, optional - you can pass the slope/radians instead, and set this to null

radians

the radians of the line, optional - you can pass the slope/degrees instead, and not set this

Variables

degrees:Float

The direction of this Ray2D, in degrees

point:Point2D

A point this Ray2D passes through. Initially set to the Point2D given in the constructor.

Changing this point's properties (or the point itself) moves the ray, while keeping its slope.

radians:Float

The direction of this Ray2D, in radians

slope:Float

The direction of this Ray2D, in a rise-over-run format.

read onlyxIntercept:Float

The x position in which y = 0

read onlyyIntercept:Float

The y position in which x = 0

Methods

distanceTo(ray:Ray2D):Float

Gets the distance between this and ray.

If this and ray intersect, 0 is returned.

Parameters:

ray

The second ray to calculate the distance to.

Returns:

A Float representing the distance. 0 if this and ray intersect.

Gets the intersection point between this and ray.

Parameters:

ray

Returns:
