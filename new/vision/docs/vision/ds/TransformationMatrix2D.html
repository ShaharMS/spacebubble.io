Used for manipulation on the Z axis, using a given point's X value. The higher a pixel's value on the Z axis, the "deeper" it is in the image, or in other terms, the closer it is to the vanishing point.

The vanishing point is decided when we apply the matrix to each pixel - it is the X and Y offsets we add to each point before the main calculation. It also plays a role in rotations and reflections, by acting like the rotation/reflection center.
