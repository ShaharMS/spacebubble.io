An integral part of the harris response function is the image's color change rate at a specific coordinate + some small value (in our case, delta). The rate of color change is the same thing as the slope of a function, hence the name of this function. The two match up well, since we only need the rate of change in either the x or y direction.

Parameters:

image

A gray scale image

coordinate

The coordinate of the pixel

delta

The small value to add to the coordinate. This can be optimized for more/less aggressive corner detection.

xDirection

Whether to use the x direction or the y direction, since we need both slopes represented (on the x and y axis)
