Vision

Vision is a cross-framework, corss platform computer vision & image manipulation library.

It offers many useful features, such as convolution, feature detection, drawing capabilities, and an Image type, compatible with any popular Haxe framework.

Drawing Learn how to draw lines, shapes & rays with Vision's Image.
Loading Images from a URL Learn how to load graphics from external sites.
Feature Detection Learn how to use Vision's implementations of popular feature detection algorithms, such as the Hough Transform, Canny Edge Detection and the Sobel Operator.
