input:

Notice, this is a very early prototype of the language. dont expect things to work correctly out of the box

Types:
  • Number, Decimal
  • Boolean
  • Letters (soon to be Characters)
  • Everything
Keywords:
  • Declare variables - define x:Type = value
  • Declare functions - action x(param:Type, anonParam) = //indent + 1
  • Write to the console - print(value)
  • Declare classes - class Name = //indent + 1
Syntax:
  • No semicolons, each line seperated by a line feed (enter) char
  • No curly brackets, each class/action seperated by indent level
  • Types are optional, but the languagee is type safe
  • Theres no null, but you have the nothing value which will throw an error when accessed
